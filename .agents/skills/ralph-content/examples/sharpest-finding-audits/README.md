# Sharpest-Finding Audits — Worked Examples

This directory contains worked examples of the Phase 1.6 Editorial Angle Audit applied to real source papers. Use these as reference cases when running the audit on new papers.

## What the audit does

Before drafting begins, enumerate 3–5 candidate findings from the source paper and score each on the 5-criterion sharpness rubric. The highest-scoring candidate becomes the editorial anchor for the post — even if it is not the paper's abstract headline.

See `SKILL.md` Phase 1.6 and `context/every-voice-patterns.md` → "The Sharpest-Finding Heuristic" for the full method.

## The 5 criteria (1–5 each, total out of 25)

1. **Counter-intuitive coefficient** — Does this finding violate what a thoughtful clinician would expect *before* reading the paper?
2. **Practical decision-changing** — Does this change how a clinician answers a real patient question or designs a protocol?
3. **Mechanism-richness** — Does this raise a "why" question worth answering, where the answer would be educational?
4. **Differentiation from default coverage** — How much of the longevity content ecosystem will lead with this finding?
5. **Specificity** — Can the finding be stated with an exact number, named comparison, or measurable contrast?

A candidate scoring **18+** is anchor-eligible. Tiebreaker: highest **counter-intuitive + practical** combined sub-score wins.

## Lessons drawn from these audits

- **The abstract is rarely the sharpest finding.** Authors optimize abstracts for citation pull. Citation pull and clinical reframing are different objectives.
- **Buried-but-sharp findings often live in the methods/results sections, not the discussion.** The Lancet paper's occupational-vs-leisure result is in section 3.4, not the abstract.
- **The "wait, why?" tell is the strongest signal.** If a finding raises a mechanism question that the post can teach into, it is almost certainly sharper than the abstract headline.
- **Scoring is a discipline, not a calculation.** The numbers don't make the decision; they force the writer to confront whether the chosen anchor *actually* differentiates from default coverage. If criterion 4 (differentiation) scores below 4, the finding is probably what every other longevity newsletter will lead with — go deeper.
